What this site is

This site introduces small tools made from real personal workflows: vocal pitch review, basic mixing guidance, tarot-style journaling, and project status tracking. Each project is designed to stay lightweight, practical, and easy to maintain.

Dev Logs

Build notes

July 6, 2026

Why I started turning apps into a mini-site

After building several small tools, the next challenge became presentation and long-term maintenance. This mini-site is designed to explain each project clearly and connect demos, code, and development notes in one place.

June 20, 2026

Separating Hub and Project Assistant roles

Nyanya Hub acts as a launcher, while Nyanya Project Assistant manages project status and logs. Keeping the tools separate makes the system easier to maintain.

June 15, 2026

Mixing Assistant QA direction

Real vocal tests showed that audio analysis tools need practical QA, not only feature development. The goal is to make feedback useful for actual mixing decisions.
