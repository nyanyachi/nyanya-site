About

Nyanya Project Lab is a collection of small AI-assisted tools and personal software projects created by Nyanyachi.

Each project is built to solve a real problem, stay lightweight, and remain easy to maintain over time.

The goal of this site is simple: to make each project easy to discover, understand, and use. Rather than combining everything into one large platform, every application remains independent while being connected through this lightweight project hub.

What I build

I build practical software for vocal practice, audio analysis, mixing assistance, tarot reading, and lightweight project management. Most projects are created to support my own daily workflows before being shared publicly.

Project philosophy

I believe useful software does not have to be large or complicated. Instead of building one massive application, I prefer creating small, focused tools that are easy to use, maintain, and improve through real-world experience.

Current focus

  • AI-assisted desktop and web tools
  • Audio analysis and creator utilities
  • Continuously improving projects through real-world use
